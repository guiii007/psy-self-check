# 心理困局自测诊断 🌿

一个模拟医院问诊流程的心理认知困局自测网页工具。

## ✨ 功能

- 📋 **标准化自测问卷**：9 题快速筛查 / 18 题标准检测，两种模式可选
- 🔍 **自动匹配诊断**：识别 7 种典型心理认知困局类型
- 💊 **对症疏导方案**：每种困局提供 4 个可落地的改善方法
- 🎨 **温和治愈风格**：暖色调 + 随机可爱背景主题 + 响应式设计
- 🎉 **答题激励**：完成答题后电子礼花庆祝 + 10 秒呼吸放松引导

## 🎯 识别困局类型

| 困局 | 说明 |
|------|------|
| 🪟 破窗效应 | 因小裂缝放任整面墙倒塌 |
| 🎯 完美主义内耗 | 高标准正在透支心理能量 |
| ⏳ 拖延型认知陷阱 | 被困在「想做但做不到」的夹缝 |
| 🌪️ 灾难化思维 | 大脑将小麻烦放大成最坏可能 |
| 🪨 习得性无助 | 被过去的失败教会了「放弃」 |
| 🎭 讨好型思维 | 照顾所有人却忘了自己 |
| ⚡ 过度焦虑思维 | 大脑马达即使不需紧张也无法停下 |

## 🚀 使用

1. 双击 `index.html` 在浏览器中打开
2. 选择「快速筛查」（9 题）或「标准检测」（18 题）
3. 逐题作答
4. 完成答题后享受礼花 + 呼吸放松
5. 查看专属诊断报告和改善方案
